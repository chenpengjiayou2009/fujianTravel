<h2>背景调研</h2>
<p>
疫情将来总会结束，世界范围内的旅游业都将迎来一波新的高潮以缓解受到疫情冲击的旅游产业。作为中国的海洋资源大省，福建省在海洋文旅方面的知名度仍需要进一步扩大。广大海外游客来中国旅行更多地仍倾向于北京，西安等历史古都。在海洋文化方面，中国福建也有着悠久的历史和传统。<br>
随着时间的推移，福建省迎来海内外的游客也只是时间问题。希望该网页能更好地向海外游客展示福建海洋文化特色以及推荐海洋相关的旅游产品。帮助福建省海洋文旅事业在世界上扩大影响力。
</p>

<h2>创意理念</h2>
<p>
以福建海洋文旅为核心的全方面多媒体旅游形象展示网页。丰富的交互体验，详尽的英文介绍可以给游客留下深刻的印象。介绍不仅包含历史背景，还包括了美景美食等广大年轻人喜闻乐见的旅行内容，交互式的旅馆推介可以给当地的旅游业经营带来直接的经济效益 。
</p>

<h3> 功能技术 </h3>
Wix 建站工具<br>
<img src="./static/wixLogo.jpg" width=100 height=50/><br>

<img src="./static/veloLogo.jpg" width=100 height=50/><br>

  无需实体服务器即可实现网页的云部署，域名为fujiantravel.co, 方便易记，部署于海外云端，海外用户访问速度更快。<br>
